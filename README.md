# Clock

## Installation

1. Clone the repository:

    ```bash
    git clone <repository-url>
    cd car-dashboard

2. Install backend dependencies:

    ```bash
    cd backend
    pip install -r requirements.txt

3. Run the application:

    ```bash
    python app.py

4. Open browser and navigate to:

    ```bash
    http://localhost:5000
